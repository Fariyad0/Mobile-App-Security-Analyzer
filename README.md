# Mobile App Security Analyzer

Static Android Application Security Assessment Framework

## Features

- Static APK Analysis
- OWASP Mobile Top 10 Mapping
- Hardcoded Secret Detection
- Exported Component Analysis
- Cleartext Traffic Detection
- Structured PDF Report Generation

## Installation

pip install flask androguard reportlab

python app.py

## Architecture

User → Web Interface → Static Analysis Engine → OWASP Mapper → PDF Report
